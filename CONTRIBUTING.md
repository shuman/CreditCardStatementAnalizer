# Contributing

Thanks for contributing to Personal Finance Intelligence.

## Development Setup

1. Create a virtual environment.
2. Install dependencies from requirements.txt.
3. Copy .env.example to .env and configure local values.
4. Run the app with bash run.sh.

## Code Style

- Keep changes focused and minimal.
- Add tests for bug fixes and new behavior.
- Avoid unrelated refactors in feature branches.

## Pull Requests

- Write clear commit messages.
- Describe what changed and why.
- Include validation steps and test results.
- Update docs when behavior or configuration changes.

## Security

If you discover a vulnerability, follow SECURITY.md instead of opening a public issue.
